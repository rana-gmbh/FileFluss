# FileFluss 0.8 Beta

## New Cloud Providers

- **WebDAV** — connect any WebDAV-compatible server (Hetzner Storage Box, ownCloud, etc.) with auto-endpoint detection and redirect-safe authentication
- **SFTP** — browse and manage files on remote servers via SSH/SFTP using the native macOS sftp binary with connection multiplexing for fast operations
- **Mega** — full integration with Mega cloud storage including file browsing, upload, download, and management
- **WordPress** — browse and manage WordPress media libraries via the REST API, organized in virtual year/month folders; supports upload, download, delete, rename, and search

## Improvements

- **Unified cloud provider picker** — redesigned account selection grid with consistent logo sizes and 4-column layout
- **Provider-aware UI** — context menus and menu bar items adapt to provider capabilities (e.g. folder creation is hidden for WordPress since folders are auto-generated by upload date)
- **Consistent logo presentation** — all provider logos now render with uniform sizing and white rounded backgrounds

## Previous Highlights (from 0.7)

- Professional conflict resolution dialog with side-by-side file comparison, newer/older labels, and Apply to All
- Smart cloud-to-cloud transfers with pre-flight conflict checking and preserved modification dates
- Universal search across local files and all connected cloud accounts
- Automatic Google Workspace conversion (Docs/Sheets/Slides to DOCX/XLSX/PPTX)
